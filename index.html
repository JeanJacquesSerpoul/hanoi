Règles du jeu

Le but du jeu est de déplacer tous les disques de la première tour vers la troisième tour.

  • On ne peut déplacer qu'un seul disque à la fois.
  • On ne peut placer un disque que sur un disque plus grand ou sur une tige vide.
  • Le jeu se termine lorsque tous les disques sont empilés sur la troisième tour.

Le nombre minimum de mouvements pour résoudre le puzzle est 2n - 1, où n est le nombre de disques.
